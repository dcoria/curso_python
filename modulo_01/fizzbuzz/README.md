 Ejercicios del curso Python y Pruebas
=========================================================

Kata FizzBuzz

Mostrar por pantalla la secuencia de números indicados. Para cada múltiplo de 3, en vez de el número, se mostrará la cadena "FIZZ", para cada múltiplo de 5 la cadena "BUZZ" y para los múltiplos tanto de 3 como de 5 la cadena "FIZZBUZZ".
