 Ejercicios del curso Python y Pruebas
=========================================================

Escribiremos un fragmento de código para obtener la información de una película a partir de su nombre utilizando el API REst: http://www.omdbapi.com/
Además implementaremos una cache con SQLite3, de manera que si la película ya está en la cache, no realiza la petición web. En caso contrario, recupera los datos de la web y los guarda en la BBDD.
